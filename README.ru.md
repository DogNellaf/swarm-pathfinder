# PSO Agent Navigation

> [🇬🇧 English](README.md) | 🇷🇺 Русский

Система **оптимизации роем частиц (PSO)**, которая подбирает оптимальные параметры навигации для жадного агента, перемещающегося по сетке с типизированной местностью.

## Возможности

- Жадная навигация агента по квадратной сетке с помощью эвристики `F(r, t, α, β) = α[t] + β / r`
- PSO-оптимизация параметров α (веса типов местности) и β (вес близости к цели)
- Настраиваемые размер роя, число итераций и коэффициенты скорости
- Утилиты генерации случайных карт и параметров
- Расчёт W без оптимизации
- Полный набор тестов на pytest

## Установка

```bash
# Клонировать репозиторий
git clone <repository-url>
cd swarm-pathfinder

# Установить зависимости для тестов
pip install pytest
```

После этого оптимизатор запускается командой:

```bash
python main.py
```

## Настройка

Все настраиваемые константы находятся в [settings.py](settings.py):

| Константа | По умолчанию | Описание |
|-----------|-------------|----------|
| `MAP_SIZE` | `10` | Длина стороны квадратной сетки |
| `LANDSCAPES_COUNT` | `10` | Число типов местности |
| `SWARM_SIZE` | `200` | Число частиц PSO |
| `STEPS_COUNT` | `1000` | Число итераций PSO |
| `CURRENT_VELOCITY_RATIO` | `0.1` | Вес инерции `w` |
| `LOCAL_VELOCITY_RATIO` | `1.0` | Когнитивный коэффициент `c1` |
| `GLOBAL_VELOCITY_RATIO` | `5.0` | Социальный коэффициент `c2` |
| `ALPHA_MIN / ALPHA_MAX` | `0 / 1` | Границы поиска для каждого `α` |
| `BETA_MIN / BETA_MAX` | `0 / 1` | Границы поиска для `β` |
| `WILL_BACKUP` | `False` | Сохранять `.backup`-копию при перегенерации данных |

## Запуск тестов

```bash
pytest tests/
```

Набор тестов охватывает вспомогательные функции, эвристики и навигацию агента, поведение отдельной частицы, полный цикл PSO и функции генерации данных.

│   └── params.json      # Начальные α, β, старт, финиш
└── tests/
    ├── conftest.py
    ├── test_utils.py
    ├── test_w_solution.py
    ├── test_particle.py
    ├── test_swarm.py
    └── test_input_generator.py
